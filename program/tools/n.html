n 切换 node 版本 #

安装 #

bash
npm i -g n

npm i -g n

为什么使用 n 切换版本无效 #

n 的安装路径和 node 默认路径不一致

  1. 查看 node 安装路径
shell
which node

which node
  1. 编辑.bash_profile文件
shell
vim ~/.bash_profile

vim ~/.bash_profile

在该文件中添加

text
export N_PREFIX=/usr/local/bin/node #根据你的安装路径而定
export PATH=$N_PREFIX/bin:$PATH

export N_PREFIX=/usr/local/bin/node #根据你的安装路径而定
export PATH=$N_PREFIX/bin:$PATH
  1. 保存并刷新文件
shell
source ~/.bash_profile

source ~/.bash_profile
  1. 重新安装 n
shell
n stable

n stable

这次就可以切换版本了，但是需要授予 sudo 权限

shell
sudo n

sudo n

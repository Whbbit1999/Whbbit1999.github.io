发布于 2022-02-23

moment 报错"Deprecation warning: value provided is not in a recognized… at Module.load (node:internal/modules/cjs/loader:975:32)"

当使用 moment 直接进行转换时，20210924160035类型的时间字符串会报错"Deprecation warning: value provided is not in a recognized… at Module.load (node:internal/modules/cjs/loader:975:32)"

可以先将传入的字符串使用构造函数进行转换，而后再进行格式化

使用 dayjs 进行转换就不会出这种错误。。。

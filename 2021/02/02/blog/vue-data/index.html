Vue组件中data为什么是函数

Vue 组件中的 data 为什么是函数？

在 vue 中组件是可以复用的，如果使用对象的方式来创建一个 data，那么后加载的 data 如果改变，会影响其他 data 中的数据。
如果使用函数的方式来创建 data，那么每个函数实例都是独立的，不会互相影响。

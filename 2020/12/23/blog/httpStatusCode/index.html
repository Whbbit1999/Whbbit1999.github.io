HTTP状态码及其含义

1开头

1XX : 信息状态码

  • 100 Continue 继续，一般在发送 post 请求时，已经发送了 http header 之后服务端将返回此信息，表示确认，之后发送具体参数信息

2开头

2XX : 成功状态码

  • 200 OK 正常返回信息
  • 201 Created 请求成功并且服务器创建了新的资源
  • 202 Accepted 服务端接受请求，但是未处理

3开头

3XX : 重定向状态码

  • 301 Moved Peramanently 请求的网页已永久移动到新位置
  • 302 Found 临时性重定向
  • 303 See Other 临时性重定向，且总是用 get 请求新的 url
  • 304 Not Modified 自上次请求后，请求的网页未修改过

4开头

4XX : 客户端错误状态码

  • 400 Bad Request 服务端无法理解客户端请求的格式，不应该以相同的格式再次发送请求到服务端
  • 401 Unauthorized 请求未授权
  • 403 Forbidden 禁止访问
  • 404 Not Found 找不到与 url 相匹配的资源

5开头

5XX : 服务端错误状态码

  • 500 Internal Server Error 最常见的服务器端错误
  • 503 Service Unavailable 服务器端暂时无法处理请求（可能是过载或维护）
